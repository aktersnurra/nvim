# nvim

My neovim config written in fennel.

# Dependencies

```
neovim = nightly
```

# Structure

```
.
└── fnl
    ├── config              # Plugin configs.
    │   └── lsp             # LSP related configs.
    │       └── settings    # Server specific settings.
    ├── plugins.fnl         # Table of all plugins.
    └── settings            # Neovim settings.
```
